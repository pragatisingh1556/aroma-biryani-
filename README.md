# Aroma Biriyani 

A full stack food ordering web application built with React, Node.js and MySQL.

![NodeJS](https://img.shields.io/badge/Node.js-green?logo=node.js)
![MySQL](https://img.shields.io/badge/MySQL-orange?logo=mysql)

## About

This is a food ordering app for a biriyani restaurant. Users can browse the menu, add items to cart, apply coupons, and place orders. There is also an admin panel to manage orders, products, users and more.

---

## Features

- User login and signup with OTP verification
- Browse menu by category and search
- Add to cart with addons
- Apply coupon codes and loyalty points
- Delivery and pickup options
- Save delivery addresses
- Track order status
- Gallery, Blog, Store Locator pages
- Admin dashboard to manage everything
- Kitchen and Delivery staff dashboards

---

## Tech Stack

- **Frontend** - React.js
- **Backend** - Node.js, Express.js
- **Database** - MySQL
- **Authentication** - JWT

---

## How to Run

**Clone the project**
```
git clone https://github.com/pragatisingh1556/aroma-biryani-.git
```

**Run backend**
```
cd server
npm install
npm start
```

**Run frontend**
```
cd client
npm install
npm start
```

**Database** - Import `database/schema.sql` in MySQL

Make sure to create a `.env` file in the server folder with your DB credentials and JWT secret.
